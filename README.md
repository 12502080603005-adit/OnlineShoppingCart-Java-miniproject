# 🛒 Online Shopping Cart - Java Mini Project

##  Project Overview

This is a **Java-based Online Shopping Cart system** developed as a mini project.
It simulates basic e-commerce functionalities such as user authentication, product management, and cart operations.

---

##  Features

*  User Registration & Login
*  Product Management (Electronics, Clothing, etc.)
*  Add to Cart / Remove from Cart
*  View Cart & Checkout
*  File Handling for Data Storage
*  Custom Exception Handling

---

##  Technologies Used

* Java (Core Java)
* OOP Concepts (Inheritance, Polymorphism, Encapsulation)
* File Handling
* Exception Handling
